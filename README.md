# Scaffold Lisk

A decentralized platform on Lisk Sepolia for creator tokens and NFTs.

## Contracts

- **CreatorToken.sol**: ERC-20 token for creators. Mints 1,000,000 tokens.
- **ArtNFT.sol**: ERC-721 NFT for art. Supports minting with base URI.
